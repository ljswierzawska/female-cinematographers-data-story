# Cinematography is a Man's World — Data Story About Female Cinematographers

This a data story about the underrepresentation of female directors of photography. Following the victory of Autumn Durald Arkapaw, this piece spotlights the rarity of nominations for women in the Best Cinematography category in the race for the Academy Awards. 

The data used in this project comes from two sources: Academy Awards Database and Celluloid Ceiling report Employment of Behind-the-Scenes Women on Top Grossing U.S. Films in
2025 produced by Dr. Martha M. Lauzen.

The data from the Academy Awards Database was scraped using Jonathan Soma’s magical scraper prompt for using an AI chatbot. It breaks down the process into steps where AI asks you follow-up question and the user needs to respond by providing it with a piece of HTML code from the data source. Link to the scraper: https://gist.github.com/jsoma/dd28675ab27c5d46b6261b93f1252307. The subsequent steps of the data analysis are conducted with Pandas Python packages. I also used Pandas for combining datapoints from teh Celluloid Ceiling. 

Data visualizations were made at first with ggplot in R and then enhanced it with Adobe Illustrator.
